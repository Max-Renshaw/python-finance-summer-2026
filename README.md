# Multi-Sector Analytics Dashboard

Welcome to my post-GCSE independent coding project. This repository contains a dual purpose python application designed to demonstrate data manipulation, predictive logic and interactive data visualisation.

## Modules
1. **Corporate Equity Analysis Tool:** Utilising 'yfinance' and 'pandas' to analyse stock volatility, market Beta and moving averages.
2. **Formula 1 Telemetry Analytics:** Utilising 'fastf1' to extract and visualise driver telemetry and throttle/brake inputs.

## Tech Stack
* Python
* Pandas (Data manipulation)
* StreamLit (Web interface)
* Plotly (Interactive charting)
